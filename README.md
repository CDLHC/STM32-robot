\# STM32F103C8T6 四足机器人

基于STM32F103C8T6开发的开源四足机器人，支持2.4G无线遥控、蓝牙控制、离线ASRPRO语音控制、多模式步态、WS2812可编程动态灯效。



\## 一、硬件清单

| 硬件组件 | 规格/数量 | 通信接口 |

| ---- | ---- | ---- |

| 主控芯片 | STM32F103C8T6 | - |

| 舵机 | SG90 ×8 | TIM2/TIM3 硬件PWM |

| 无线遥控模块 | NRF24L01 | SPI2 |

| 离线语音模块 | ASRPRO | USART3 |

| RGB灯带 | WS2812B ×8 | TIM4\_CH2 + DMA |

| 显示屏 | 0.96" OLED 128×64 | I2C |

| 蜂鸣器 | 有源蜂鸣器 ×1 | GPIO |

| 蓝牙串口模块 | HC-05 | USART1 |

\## 三、步态模式功能

内置全套可切换动作步态：

1\. 基础移动：前进、后退、左转、右转

2\. 原地旋转：顺时针自转、逆时针自转

3\. 静态姿态：坐下

4\. 演示动作：自动跳舞模式

5\. 感应交互：拿起检测（触发蜂鸣器报警+膝盖舵机抖动）



\## 四、FreeRTOS 软件任务架构

采用FreeRTOS分时多任务调度，各功能独立任务、互不阻塞：

├── ServoTask   (High)   20ms  舵机角度 PWM 输出刷新

├── GaitTask    (Normal) 25ms  步态状态机运算

├── SerialTask  (Normal) 25ms  蓝牙 / 语音串口指令解析

├── SensorTask  (Normal) 30ms  机身拿起检测采样

├── LedTask     (Low)    8ms   WS2812 动态灯效渲染

├── NRFTask     (Normal) 25ms  NRF24L01 无线数据包接收

└── DefaultTask (Normal) —     开机初始化灯光动画



\## 五、串口控制指令表

单字节十六进制指令，蓝牙/语音/无线三通道通用：

| 指令码 | 控制功能 |

| ---- | ---- |

| 0x00 | 整机停止，复位初始姿态 |

| 0x01 | 前进 |

| 0x06 | 后退 |

| 0x02 | 右转 |

| 0x03 | 左转 |

| 0x04 | 顺时针原地旋转 |

| 0x05 | 逆时针原地旋转 |

| 0x10 | 切换坐下姿态 |

| 0x08 | 自动演示跳舞步态 |

| 0x09 | 开启彩虹渐变灯效 |

| 0x21 | 关闭全部灯带 |



\## 六、开发环境

\- 编译IDE：Keil MDK-ARM / STM32CubeIDE

\- 主控MCU：STM32F103C8T6

\- 底层驱动库：STM32Cube HAL FW\_F1

\- 实时操作系统：FreeRTOS CMSIS-RTOS v1



\## 七、项目核心特性

1\. 无线、语音、蓝牙三路指令互通，任意设备均可控制机器人；

2\. WS2812灯带依靠DMA硬件输出时序，不占用CPU运算资源；

3\. 8路舵机全部使用硬件定时器PWM，无软件延时造成的抖动；

4\. 机身拿起检测功能，可实现防跌落、防误操作报警；

5\. 模块化FreeRTOS架构，代码分层清晰，便于二次开发与功能扩展。
